Ramas

Las ramas son utilizadas para desarrollar funcionalidades aisladas unas de otras. La rama master es la rama por "defecto" cuando creas un repositorio. Usa otras ramas para el desarrollo y fusionalas de regreso a la rama principal al terminar. para crear una rama ejecuta el siguiente comando:
  • git -b rama
para cambiarte a esa rama ejecuta el siguiente comando:
  • git checkout rama
    • para borrar una rama usa siguiente comando:
    • git branch -d rama
    para fusionar una rama usa siguiente comando:
    • git merge rama
    comparte tu rama:
    • git push origin rama
    descarga las ramas de tus amigos:
    • git checkout -b rama origin/rama
    borrar una rama remota:
    • git push origin :rama
